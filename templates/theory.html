Machine Learning Models

Understanding the algorithms behind our cardiovascular risk assessment tool.

Logistic Regression

Logistic Regression is a foundational statistical model used for binary classification tasks. It works by applying a Sigmoid function to a linear combination of input features (like age, cholesterol, and blood pressure), mapping the output to a probability score between 0 and 1.

Characteristics

  • Highly interpretable (clear feature importance).
  • Assumes a linear relationship between features and the log-odds of the target.
  • Can struggle with complex, non-linear patterns often found in clinical data.

Decision Tree

Decision Trees classify data by recursively splitting it based on the feature that provides the highest information gain. It creates a flowchart-like structure where each internal node represents a test on a clinical parameter, and each leaf node represents the prediction.

Characteristics

  • Excellent at capturing non-linear relationships.
  • Easy to visualize and explain to medical professionals.
  • Highly prone to overfitting the training data, leading to high variance and poorer generalization on new patients.
Selected Model

Random Forest Ensemble

Random Forest is an ensemble learning method that constructs a multitude of decision trees during training. For classification tasks like ours, it outputs the mode of the classes (the majority vote) predicted by individual trees.

Why We Chose Random Forest

Variance Reduction

By averaging multiple trees built on random subsets of data, it significantly mitigates the overfitting problem inherent to single Decision Trees.

Complex Interactions

It inherently captures complex, non-linear interactions among clinical variables (e.g., how high BMI interacts with high blood pressure) better than Logistic Regression.

Superior Generalization for Healthcare

For structured, tabular healthcare datasets, Random Forests consistently provide a superior balance of high accuracy and robust generalization to new, unseen patient data.
